# 🎌 AnimeVerse

AnimeVerse is a modern and responsive anime discovery web app built using HTML, CSS, and JavaScript. It uses the Jikan API to fetch real-time anime data and provides a smooth browsing experience across all devices.

---

## 🚀 Features

- 🔎 Instant anime search
- 🔥 Trending & popular anime lists
- 🎭 Genre-based filtering
- 📄 Detailed anime modal view
- 🌙 Dark / Light mode toggle
- 📱 Fully responsive design (mobile, tablet, desktop)

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (ES6+)
- Jikan API

---

## 📡 API Used

- Jikan API: https://api.jikan.moe/v4

---

## 📱 Responsive Support

- Mobile devices
- Tablets
- Laptops & desktops
- Foldable screens
